Concept 3 / The Signal

A navigation-aware version of the current canvas where remote behaviour is visible enough to trust, but not so loud that it stops feeling like design work.

The Signal is for the current app if remote simulation becomes a more central promise. It overlays focus connections, default entry, and route summaries directly onto the canvas, then uses the right panel for detailed neighbour editing and action checks.
