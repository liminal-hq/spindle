Concept 1 / The Harbour

A calm main editor where selecting a button opens a docked behaviour tray directly under the canvas instead of sending the user into Bind or Remote mode.

The Harbour keeps the current Spindle layout, but the selected node becomes the centre of gravity. Action binding, route targets, stream actions, and immediate compile implications all appear in a tray that belongs to the active selection, not to a separate tab.
