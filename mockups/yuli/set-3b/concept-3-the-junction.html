Concept 3 / The Junction

A more power-user integrated editor with a permanent interaction rail inside the main view: actions, focus paths, generator lineage, and compile risk all visible at once.

The Junction assumes some users would rather keep everything exposed if it means fewer mode switches. It integrates the binding table, remote summary, and compile audit into one operational rail that sits below the artboard and updates live as the selection changes.
