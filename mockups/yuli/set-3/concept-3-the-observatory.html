Concept 3 / The Observatory

A navigation control room where focus routes, default entry points, timeout flows, and compile integrity are always in view.

Some discs succeed or fail on behaviour more than visual polish. The Observatory is for those moments. It keeps the authored scene visible, but it elevates the interaction graph to first-class status so broken neighbours, unreachable nodes, and confusing returns are caught early.
