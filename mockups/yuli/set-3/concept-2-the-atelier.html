Concept 2 / The Atelier

A craft studio where themes, reusable button families, and motion styling feel like material choices instead of hidden configuration.

The Atelier leans into the authored-scene future. It makes components, theme tokens, generated layouts, and stateful button treatments feel tactile. The user is still designing real DVD navigation, but the surface invites them to think in systems and moods rather than one-off rectangles.
